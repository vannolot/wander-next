# WanderNext

**Explore directions** — one-page website.

## Struktura

```
wandernext/
├── index.html          # Główny plik HTML
├── css/
│   └── styles.css      # Wszystkie style
├── js/
│   └── main.js         # Scroll nav + fade-up animations
└── img/
    ├── logo.svg             # Logo (czarne, do nawigacji)
    ├── logo-white.svg       # Logo (białe, do footera)
    ├── hero-grid.svg        # Abstrakcyjna siatka kompassu w hero
    └── diagram-compass.svg  # Diagram kierunków w sekcji Studio
```

## Sekcje

1. **Hero** — headline z gradientem, orby kolorowe, siatka kompasu
2. **Intro strip** — podejście / model / specjalizacja
3. **01 Studio** — opis agencji + diagram kierunków
4. **02 Podejście** — Exploration → Direction → Execution
5. **03 Obszary** — 4 karty usług
6. **04 Oferta** — 4 moduły współpracy
7. **Manifesto** — cytat
8. **05 Kontakt** — CTA + email
9. **Footer**

## Fonty

Ładowane z Google Fonts:
- **DM Serif Display** — nagłówki
- **DM Sans** — body text

## GitHub Pages

Po wrzuceniu na repo wystarczy włączyć GitHub Pages z brancha `main` i folderu `/ (root)`.
